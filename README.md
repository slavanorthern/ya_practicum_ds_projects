Репозиторий содержит проекты, выполненные в рамках курса Data Science  от Яндекс Практикум.

| Финальный проект по телекоммуникации | Оператор связи хочет научиться прогнозировать отток клиентов. Если выяснится, что пользователь планирует уйти, ему будут предложены промокоды и специальные условия. Команда оператора собрала персональные данные о некоторых клиентах, информацию об их тарифах и договорах. | Python, Pandas, Numpy, Matplotlib, Seaborn, Scikit-Learn | 
| Сборный проект 1. Исследование игр для интернет-магазина  | Из открытых источников доступны исторические данные о продажах игр, оценки пользователей и экспертов, жанры и платформы.
